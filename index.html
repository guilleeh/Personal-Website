    Contact Keeper

    MERN Stack application that allows you to create and manage contacts

    Developed in Summer 2019, Contact Keeper is an application that allows you to create an account and keep track of all your contacts. Contacts are maintained in a database and all CRUD operations are available to be performed on your contacts. This was my very first attempt at creating a fully fledged out full stack application.

    OC United

    React Application for Non-profit OC United

    Developed during Spring Quarter of 2018, the OC United Website was created by a group of 6 UCI students. I led backend development of the site. We used Firebase realtime database along with Node, express and stripe to handle user event registration and donation processing.

    Github Finder

    Web Application used to search GitHub users

    GitHub Finder was my first attempt at creating a React application with the use of the Context API. This way, I was able to pass state to components without having to pass down props. It was a great learning experience which led to cleaner and more consice code.
